Command line Cheat Sheet

mkdir new-directory

make a new directory, called 'new-directory'

touch

create file, eg touch index.html

ls

Lists files in a directory

cd new-directory

Stands for Change Directory, moves into directory called 'new-directory'

cd ..

Move up to the level above

cd -

move you back to the last directory you were in

pwd

stands for Print Working Directory, prints out the directory you are currently in

man

Stands for Manual, print infront of any command you don't know to see the full documentation. eg man pwd

caffinate

Prevents your laptop from turning off or sleeping. To stop is simple press Control-C

ping

Ping allows you to check the server response time to a specific URL. If you think a website is down, this can be a really useful way to double check it isn’t just your computer crashing e.g ping www.google.com.

cat FILE

This prints out the text of a file onto your terminal, which can be useful if you are searching for something, this is especially useful with the grep command.

grep

searches any given input files for specific patterns, and will print them out to the terminal for you

cat FILE | grep and

Here you can use a pipe, the long vertical line, to append two arguments together, so this command prints out the contents of the file, but only those lines with ‘and’ in them.

say superhi is great!

If you’re feeling lonely, you can always chat to your command line!
